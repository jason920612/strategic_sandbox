# RFC-090：分階段開發路線

- Status: Draft
- Type: Roadmap
- Scope: 多階段開發
- Version: 0.1

## 1. 原則

本遊戲必須拆成很多階段。這份路線圖先列出大約 120 個可執行任務。正式開發時可以繼續拆到數百個任務。

目標：能啟動、讀資料、跑時間、輸出 log。

## 2. 第一個推薦開發衝刺

最先做：

1. Milestone 0.1–0.13
2. Milestone 1.1–1.17
3. Milestone 2.1–2.8
4. Milestone 4.1–4.6

完成後應該能：

- 選一國
- 改預算
- 跑時間
- 看派系與穩定變化
- 匯出 SVG
- 匯出 CSV

# RFC-000：遊戲總覽與設計憲章

- Status: Draft
- Type: GDD / Design Constitution
- Scope: 全專案
- Version: 0.1

## 1. 遊戲定位

本遊戲是一款 1930–2000 全球政治經濟戰略沙盒。玩家扮演國家最高領導人，而不是全知全能的上帝或微操軍隊的總司令。

核心不是重演歷史，而是讓世界透過制度、派系、經濟、恐懼、利益、誤判與隨機權重自然演化。

## 2. 核心一句話

玩家制定政治、法律、預算、外交與戰略目標；國家機器、軍方、官僚、情報部門、媒體、地方勢力與外國政府會根據自身利益自動執行、拖延、扭曲或背叛。

## 3. 時間與地圖

- 時間：1930–2000，可超過 2000 繼續
- 地圖：全球，省份／州級
- 時間制：即時暫停制
- 底層 tick：每日
- 長期更新：週、月、年分層處理

## 4. 玩家身份

玩家是國家領導人。玩家能失勢、被架空、入獄、流亡或成為內戰一方。只要玩家本人未死亡，遊戲可以繼續。

玩家死亡時遊戲結束；旁觀模式則可切換國家。

## 5. 不可違背設計原則

1. 玩家是領導人，不是全知全能的神。
2. 命令不等於結果，國家機器可能拖延、扭曲、抵抗或背叛。
3. 世界不靠強劇本推進，而靠利益、恐懼、誤判、制度與隨機權重自然演化。
4. 玩家看到的資訊不一定是真相。
5. 軍隊、情報、媒體、官僚、地方勢力都是半獨立行動者。
6. 政權倒台不一定 Game Over；玩家死亡才是 Game Over。
7. 原型必須極小，先驗證政策、派系、穩定、GDP、稅收、預算。
8. 每個開發階段都必須能獨立跑模擬、輸出 log、驗證結果。
9. 任何複雜公式都必須能降級成可除錯的簡化模型。
10. 所有隨機都必須經 seed 控制，長期目標是可重播。
